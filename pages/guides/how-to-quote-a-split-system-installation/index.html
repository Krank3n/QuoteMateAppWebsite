Quoting Guide — HVAC

How to Quote a Split System Installation in Australia

Guide for HVAC technicians on quoting split system air conditioning installations. Covers unit selection, piping, electrical, and installation costs.

What to Include in a Split System Quote

A split system installation quote should cover: indoor and outdoor unit (brand, model, capacity), wall bracket for outdoor unit, refrigerant piping (length and diameter), electrical cable and dedicated circuit, drain pipe, pipe cover/ducting (cosmetic), and commissioning/testing. Always specify the unit brand, model, and kW capacity.

Split System Costs in 2026

2.5kW unit (small bedroom): $1,200–$1,800 installed. 5kW unit (living room): $1,800–$2,800 installed. 7kW unit (large living area): $2,200–$3,500 installed. 8.5kW+ unit (open plan): $3,000–$4,500 installed. These prices include supply, back-to-back installation, and electrical. Long pipe runs or difficult access add $200–$800.

Pipe Run and Electrical Considerations

Standard back-to-back installation (indoor and outdoor units on the same wall) keeps pipe runs short and costs down. Longer pipe runs (up to 20m) require more refrigerant copper piping and increase cost by $30–$50 per additional metre. Electrical: most split systems need a dedicated circuit from the switchboard — check if capacity exists.

Sizing the Right Unit

Undersized units work harder, cost more to run, and fail sooner. Oversized units short-cycle and don't dehumidify properly. Calculate capacity based on room size, insulation, window area, and orientation. A general guide: 120–150W per sqm for well-insulated rooms, 150–200W per sqm for poorly insulated rooms. Always recommend the right size, even if it's more expensive.

Quoting Tips for HVAC

1. Conduct a site visit to assess mounting positions and pipe run. 2. Check switchboard capacity for the new circuit. 3. Offer multiple unit options (good, better, best). 4. Include warranty details in the quote. 5. Note the energy star rating and running costs to help customers decide.

Quick Tips Checklist

  • Conduct a site visit to plan unit positions and pipe run
  • Check switchboard capacity — an upgrade may be needed
  • Offer 2–3 unit options at different price points
  • Include the energy star rating and estimated running costs
  • Quote pipe covers as an optional extra for a neat finish
  • Specify warranty periods for unit, compressor, and labour separately
